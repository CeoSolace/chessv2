# ChessV2

Guest-only online chess.

## Features

- No accounts
- No database
- Create room
- Join room with code
- Online 1v1
- Spectators after 2 players
- Legal move validation with chess.js
- Render-ready

## Local

```bash
npm install
npm start
```

Open `http://localhost:3000`.

## Render

Build command:

```bash
npm install
```

Start command:

```bash
npm start
```
